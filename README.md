# Curio

Приложение для управления расписанием занятий детей с преподавателями.

## Технологии

- **Qt 6.8** - фреймворк для разработки кроссплатформенных приложений
  - Qt Quick - для создания современного пользовательского интерфейса
  - Qt QuickControls2 - набор готовых компонентов UI
  - Qt Widgets - дополнительные виджеты
- **C++** - основной язык программирования
- **QML** - декларативный язык для создания интерфейса
- **CMake** - система сборки проекта

## Функции

- **Управление детьми** - добавление и редактирование профилей детей
- **Управление преподавателями** - ведение базы преподавателей
- **Управление уроками** - создание и планирование занятий
- **Повторяющиеся уроки** - поддержка регулярных занятий по расписанию
- **Календарь** - просмотр расписания по дням и неделям
- **Экспорт данных** - возможность экспорта информации
- **Хранение данных** - автоматическое сохранение и загрузка данных
- **Авторизация** - система входа в приложение
- **Настройки** - конфигурация приложения

## Запуск проекта

### Требования

- Qt 6.8 или выше
- CMake 3.16 или выше
- Компилятор C++ с поддержкой стандарта C++17

### Сборка и запуск

1. Клонируйте репозиторий:
```bash
git clone <url-репозитория>
cd curio
```

2. Создайте директорию для сборки:
```bash
mkdir build
cd build
```

3. Сконфигурируйте проект с помощью CMake:
```bash
cmake ..
```

4. Соберите проект:
```bash
cmake --build .
```

5. Запустите приложение:
```bash
./appcurio
```

### Примечание

При закрытии приложения все данные автоматически сохраняются и загружаются при следующем запуске.
